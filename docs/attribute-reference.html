Attribute Reference

Attribute description

Attribute descriptions are in the following format: section.attributename, valuetype, short attribute description

Here follows a list of defined attribute valuetypes used in the reference documentation:

bool, a string value of true or false

integer, a signed integer value

string, a string value

xyzid_, specifies an id reference of type xyz

duration, durations can be specified in seconds and milliseconds with integer suffix s, ms (eg. 20s, 20000ms)

[val1 : val2], this describes a list of fixed string values for attribute

[val1 (xyzid), val2 (duration), val3 (integer)]_, comma seperated values ex: 43,4s,200

[val1 (string), ...], comma separated list of n values

float, a floating point number

alignment, can be any of: topleft, top, topright, left, center, right, bottomleft, bottom, bottomright

label, specified as: "hidden" or as: x (integer), y (integer), justify (string), valign (string), style (string); justify can be [left:center:right]; valign can be [top:center:bottom]; style is any font style defined in engine/font_settings.txt

direction, directions can be N, NE, E, SE, S, SW, W, NW, or an integer value in the range of [0-7] inclusive

Animation

Description of animations in animations/

image, _string_, Filename of sprite-sheet image.

position, _integer_, Number of frames to the right to use as the first frame. Unpacked animations only.

frames, _integer_, The total number of frames

duration, _integer_, The duration of the entire animation in 'ms' or 's'.

type, _[play_once, back_forth, looped]_, How to loop (or not loop) this animation.

render_size, _w (integer), h (integer)_, Width and height of animation.

render_offset, _x (integer), y (integer)_, Render x/y offset.

active_frame, _[all:frame (integer), ...]_, A list of frames marked as "active". Also, "all" can be used to mark all frames as active.

frame, _index (integer), direction (integer), x (integer), y (integer), w (integer), h (integer), x offset (integer), y offset (integer)_, A single frame of a compressed animation.

EffectManager

Description of "type" in powers/effects.txt

damage, Damage per second

damage_percent, Damage per second (percentage of max HP)

hpot, HP restored per second

hpot_percent, HP restored per second (percentage of max HP)

mpot, MP restored per second

mpot_percent, MP restored per second (percentage of max MP)

speed, Changes movement speed. A magnitude of 100 is 100% speed (aka normal speed).

immunity, Removes and prevents bleed, slow, stun, HP/MP steal, and damage reflection. Magnitude is ignored.

stun, Can't move or attack. Being attacked breaks stun.

revive, Revives the player. Typically attached to a power that triggers when the player dies.

convert, Causes an enemy or an ally to switch allegiance

fear, Causes enemies to run away

knockback, Pushes the target away from the source caster. Speed is the given value divided by the framerate cap.

offense, Increase Offense stat.

defense, Increase Defense stat.

physical, Increase Physical stat.

mental, Increase Mental stat.

$STATNAME, Increases $STATNAME, where $STATNAME is any of the base stats. Examples: hp, dmg_mele_min, xp_gain

$ELEMENT_resist, Increase Resistance % to $ELEMENT, where $ELEMENT is any found in engine/elements.txt. Example: fire_resist

death_sentence, Causes sudden death at the end of the effect duration.

shield, Create a damage absorbing barrier based on Mental damage stat. Duration is ignored.

heal, Restore HP based on Mental damage stat.

EventManager

Description of events in maps/ and npcs/

event.type, _[on_trigger:on_mapexit:on_leave:on_load:on_clear]_, Type of map event.

event.location, _[x,y,w,h]_, Defines the location area for the event.

event.hotspot, _[ [x, y, w, h] : location ]_, Event uses location as hotspot or defined by rect.

event.cooldown, _duration_, Duration for event cooldown in 'ms' or 's'.

event.reachable_from, _[x,y,w,h]_, If the hero is inside this rectangle, they can activate the event.

event.tooltip, _string_, Tooltip for event

event.power_path, _[hero:[x,y]]_, Event power path

event.power_damage, _min(integer), max(integer)_, Range of power damage

event.intermap, _[map(string),x(integer),y(integer)]_, Jump to specific map at location specified.

event.intramap, _[x(integer),y(integer)]_, Jump to specific position within current map.

event.mapmod, _[string,int,int,int],.._, Modify map tiles

event.soundfx, _[soundfile(string),x(integer),y(integer),loop(boolean)]_, Filename of a sound to play. Optionally, it can be played at a specific location and/or looped.

event.loot, _[string,drop_chance([fixed:chance(integer)]),quantity_min(integer),quantity_max(integer)],..._, Add loot to the event; either a filename or an inline definition.

event.loot_count, _min (integer), max (integer)_, Sets the minimum (and optionally, the maximum) amount of loot this event can drop. Overrides the global drop_max setting.

event.msg, _string_, Adds a message to be displayed for the event.

event.shakycam, _duration_, Makes the camera shake for this duration in 'ms' or 's'.

event.requires_status, _string,..._, Event requires list of statuses

event.requires_not_status, _string,..._, Event requires not list of statuses

event.requires_level, _integer_, Event requires hero level

event.requires_not_level, _integer_, Event requires not hero level

event.requires_currency, _integer_, Event requires atleast this much currency

event.requires_not_currency, _integer_, Event requires no more than this much currency

event.requires_item, _integer,..._, Event requires specific item

event.requires_not_item, _integer,..._, Event requires not having a specific item

event.requires_class, _string_, Event requires this base class

event.requires_not_class, _string_, Event requires not this base class

event.set_status, _string,..._, Sets specified statuses

event.unset_status, _string,..._, Unsets specified statuses

event.remove_currency, _integer_, Removes specified amount of currency from hero inventory

event.remove_item, _integer,..._, Removes specified item from hero inventory

event.reward_xp, _integer_, Reward hero with specified amount of experience points.

event.reward_currency, _integer_, Reward hero with specified amount of currency.

event.reward_item, _x(integer),y(integer)_, Reward hero with y number of item x.

event.restore, _string_, Restore the hero's HP, MP, and/or status.

event.power, _power_id_, Specify power coupled with event.

event.spawn, _[string,x(integer),y(integer)], ..._, Spawn an enemy from this category at location

event.stash, _boolean_, Opens the Stash menu.

event.npc, _string_, Filename of an NPC to start dialog with.

event.music, _string_, Change background music to specified file.

event.cutscene, _string_, Show specified cutscene by filename.

event.repeat, _boolean_, Allows the event to be triggered again.

event.save_game, _boolean_, Saves the game when the event is triggered. The respawn position is set to where the player is standing.

event.book, _string_, Opens a book by filename.

ItemManager: Items

Description about the class and it usage, items/items.txt...

id, _integer_, An uniq id of the item used as reference from other classes.

name, _string_, Item name displayed on long and short tooltips.

flavor, _string_, A description of the item.

level, _integer_, The item's level. Has no gameplay impact. (Deprecated?)

icon, _integer_, An id for the icon to display for this item.

book, _string_, A book file to open when this item is activated.

quality, _string_, Item quality matching an id in items/qualities.txt

item_type, _string_, Equipment slot [artifact, head, chest, hands, legs, feets, main, off, ring] or base item type [gem, consumable]

equip_flags, _flag (string), ..._, A comma separated list of flags to set when this item is equipped. See engine/equip_flags.txt.

dmg_melee, _[min (integer), max (integer)]_, Defines the item melee damage, if only min is specified the melee damage is fixed.

dmg_ranged, _[min (integer), max (integer)]_, Defines the item ranged damage, if only min is specified the ranged damage is fixed.

dmg_ment, _[min (integer), max (integer)]_, Defines the item mental damage, if only min is specified the ranged damage is fixed.

abs, _[min (integer), max (integer)]_, Defines the item absorb value, if only min is specified the absorb value is fixed.

requires_stat, _[ [physical:mental:offense:defense], amount (integer) ]_, Make item require specific stat level ex. requires_stat=physical,6 will require hero to have level 6 in physical stats

requires_class, _string_, The hero's base class (engine/classes.txt) must match for this item to be equipped.

bonus, _[stat_name (string), amount (integer)]_, Adds a bonus to the item power_tag being a uniq tag of a power definition, e.: bonus=HP regen, 50

soundfx, _string_, Sound effect filename to play for the specific item.

gfx, _string_, Filename of an animation set to display when the item is equipped.

loot_animation, _filename (string), min quantity (int), max quantity (int)_, Specifies the loot animation file for the item. The max quantity, or both quantity values, may be omitted.

power, _power_id_, Adds a specific power to the item which makes it usable as a power and can be placed in action bar.

replace_power, _old (integer), new (integer)_, Replaces the old power id with the new power id in the action bar when equipped.

power_desc, _string_, A string describing the additional power.

price, _integer_, The amount of currency the item costs, if set to 0 the item cannot be sold.

price_sell, _integer_, The amount of currency the item is sold for, if set to 0 the sell prices is prices*vendor_ratio.

max_quantity, _integer_, Max item count per stack.

pickup_status, _string_, Set a campaign status when item is picked up, this is used for quest items.

stepfx, _string_, Sound effect when walking, this applies only to armors.

disable_slots, _type (string), ..._, A comma separated list of equip slot types to disable when this item is equipped.
